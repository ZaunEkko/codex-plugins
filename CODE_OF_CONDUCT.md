# Code of Conduct

[简体中文](CODE_OF_CONDUCT.md) · [English](i18n/en/CODE_OF_CONDUCT.md) · [繁體中文](i18n/zh-TW/CODE_OF_CONDUCT.md) · [日本語](i18n/ja/CODE_OF_CONDUCT.md) · [한국어](i18n/ko/CODE_OF_CONDUCT.md)

## 我们的承诺

为了营造开放、友好、尊重的社区环境，参与者和维护者承诺让本项目成为一个没有骚扰和歧视的协作空间。

## 鼓励的行为

- 使用友好、包容、尊重的语言。
- 尊重不同观点和经验。
- 给出建设性反馈，聚焦问题和改进。
- 在讨论中优先考虑项目目标和使用者体验。

## 不接受的行为

- 人身攻击、辱骂、骚扰、威胁或持续打扰。
- 基于身份、地区、语言、性别、年龄、能力等因素的歧视性言论。
- 未经允许公开他人隐私信息。
- 恶意刷屏、引战、偏离主题或破坏协作氛围的行为。

## 执行方式

维护者可以根据情况提醒、编辑或删除不当内容、关闭讨论、限制参与者继续互动，或采取其他合理措施保护社区协作环境。

## 适用范围

本准则适用于本仓库的 issue、Pull Request、discussion、代码评审以及其他与项目相关的公共协作场景。
