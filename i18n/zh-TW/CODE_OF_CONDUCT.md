# Code of Conduct

[简体中文](../../CODE_OF_CONDUCT.md) · [English](../en/CODE_OF_CONDUCT.md) · [繁體中文](CODE_OF_CONDUCT.md) · [日本語](../ja/CODE_OF_CONDUCT.md) · [한국어](../ko/CODE_OF_CONDUCT.md)

## 我們的承諾

為了營造開放、友善、尊重的社群環境，參與者和維護者承諾讓本專案成為沒有騷擾和歧視的協作空間。

## 鼓勵的行為

- 使用友善、包容、尊重的語言。
- 尊重不同觀點和經驗。
- 給出建設性回饋，聚焦問題和改進。
- 在討論中優先考慮專案目標和使用者體驗。

## 不接受的行為

- 人身攻擊、辱罵、騷擾、威脅或持續打擾。
- 基於身份、地區、語言、性別、年齡、能力等因素的歧視性言論。
- 未經允許公開他人隱私資訊。
- 惡意刷屏、引戰、偏離主題或破壞協作氛圍的行為。

## 執行方式

維護者可以根據情況提醒、編輯或刪除不當內容、關閉討論、限制參與者繼續互動，或採取其他合理措施保護社群協作環境。
