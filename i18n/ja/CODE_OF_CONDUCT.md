# Code of Conduct

[简体中文](../../CODE_OF_CONDUCT.md) · [English](../en/CODE_OF_CONDUCT.md) · [繁體中文](../zh-TW/CODE_OF_CONDUCT.md) · [日本語](CODE_OF_CONDUCT.md) · [한국어](../ko/CODE_OF_CONDUCT.md)

## 私たちの誓約

開かれた、友好的で尊重のあるコミュニティを育てるため、参加者とメンテナーは、このプロジェクトをハラスメントや差別のない協働の場にすることを約束します。

## 推奨される行動

- 歓迎的で、包容的で、敬意のある言葉を使う。
- 異なる視点や経験を尊重する。
- 問題と改善に焦点を当てた建設的なフィードバックを行う。
- 議論ではプロジェクトの目的と利用者体験を優先する。

## 受け入れられない行動

- 個人攻撃、侮辱、ハラスメント、脅迫、継続的な妨害。
- アイデンティティ、地域、言語、性別、年齢、能力などに基づく差別的な発言。
- 許可なく他人の個人情報を公開すること。
- スパム、煽り、脱線、協働環境を損なう行為。

## 執行

メンテナーは状況に応じて、注意、不適切な内容の編集または削除、議論の終了、参加制限、その他コミュニティを守るための合理的な措置を取ることがあります。
